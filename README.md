# Telecom_Customer_churn_analysis
 data Analysis Project on Telecom Customer Churn using SQL, python, Excel, and Power BI Dashboards.

# Project Overview 
This project analyzes telecom customer data to understand customer churn behavior and identify factors that lead to 
customer loss.

# Dataset 
Telco Customer Churn Dataset 
• Total Customers: 7043 
• Churn Customers: 1869 
• Churn Rate: 27% 

# Tools Used 
• Excel 
• SQL 
• Python 
• Power BI 

# Key Insights 
• Month-to-Month contract customers have the highest churn rate. 
• Customers using Electronic Check payment method churn more frequently. 
• Customers with low tenure (less than 12 months) are more likely to churn. 
• Customers with higher monthly charges show higher churn probability. 

# Business Impact 
Customer churn reduces company revenue and increases the cost of acquiring new customers. 

# Recommendations 
• Encourage customers to switch to long-term contracts. 
• Promote automatic payment methods such as credit card or bank transfer. 
• Improve customer experience during the first year of service. 
• Focus retention strategies on high-risk customers.
